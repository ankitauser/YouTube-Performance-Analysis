# YouTube Channel Performance Analysis

## Project Overview
Analyzed YouTube channel performance data to identify key factors driving revenue and viewer engagement. Built and compared Machine Learning models to predict estimated revenue.

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Scikit-learn (Random Forest, Gradient Boosting)
- SHAP (Model Interpretability)
- Google Colab

## Dataset
YouTube channel real performance analytics data including views, likes, shares, video duration, average view duration, and estimated revenue.

## Steps Performed
1. Data Loading & Preview
2. Feature Engineering (time-based features, Engagement Quality metric)
3. Exploratory Data Analysis (EDA)
4. Model Comparison — Random Forest vs Gradient Boosting
5. Feature Importance Analysis
6. SHAP Analysis for model interpretability
7. Partial Dependence Plots

## Key Results
- Random Forest and Gradient Boosting models compared using RMSE
- Top revenue-driving features identified using SHAP analysis
- Views and Engagement Quality found to be strongest predictors
